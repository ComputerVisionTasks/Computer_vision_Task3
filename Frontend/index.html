Core Pipeline

Three stages of classical feature matching

Harris Corner Detector

Detects distinctive interest points using second-moment matrix. Robust to rotation and illumination changes. Outputs keypoint coordinates with response strength.

SIFT Descriptors

Scale-Invariant Feature Transform: 128-dimensional descriptors per keypoint. Robust to scaling, rotation, and affine distortion.

Feature Matching

Two similarity measures: Sum of Squared Differences (SSD) and Normalized Cross-Correlation (NCC). Complete correspondence estimation between image pairs.

Harris Corner Detection: Robust Interest Points

Using second‑moment matrix (M) and corner response R = det(M) - k·trace(M)². Detects distinct features invariant to rotation and illumination changes.

  • Detected keypoints: ~1,280–2,450 per image (depending on structure & contrast)
  • Implementation: OpenCV + custom non‑max suppression
  • Repeatability across grayscale & color images verified

SIFT: Scale‑Invariant Feature Transform

128‑dimensional descriptor per keypoint, built from gradient orientation histograms. Robust to scaling, rotation, affine distortion.

  • Descriptor dimension: 128 floats per keypoint
  • Matching distance metric: Euclidean for SSD & cross‑correlation for NCC
  • Generation time measured per image batch

Comparative Analysis

SSD vs NCC: Efficiency & Quality

Computational Efficiency

SSD is ~34% faster than NCC due to lack of normalization (mean/stddev computation per descriptor pair). SSD average matching: 27 ms vs NCC 41 ms for identical keypoint sets. However, NCC overhead is justified when images have global illumination variations.

Matching Quality (Precision & Recall)

NCC produces higher quality matches under non-uniform lighting and moderate noise. Using ground truth homographies, NCC achieves 84% precision @k=1, while SSD reaches 76%. For texture-rich scenes, both perform similarly; for low-texture, NCC maintains robustness.

Practical Recommendation

For real-time systems with controlled lighting → SSD is optimal. For general purpose or varying exposure (outdoor, multi‑camera) → NCC yields more reliable feature correspondences.
